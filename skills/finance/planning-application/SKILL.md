# Planning Application

## When to use this skill
- 當使用者說「幫我排申請時程」
- 當需要追蹤多所學校的申請進度
- 當需要規劃 IELTS/GRE 備考時間
- 當需要準備 SOP、推薦信、作品集等文件

## Workflow

### 1. Plan
- 確認目標入學時間（如 2028 Q1）
- 確認目標學校清單（3-6 所，分 reach/match/safety）
- 反推準備時間軸（通常需提前 12-18 個月）

### 2. Map Timeline
- 標註每所學校的申請截止日
- 標註考試報名截止日和出分時間
- 標註推薦信/SOP/CV 的準備節點
- 預留 buffer（考試可能需要重考）

### 3. Track
- 建立 checklist 追蹤每個項目的完成狀態
- 設定里程碑提醒
- 每月 review 一次進度

### 4. Adapt
- 若考試成績不如預期，啟動 B 計劃（重考/換校）
- 若獎學金申請需額外文件，及時補充
- 若簽證政策變動，重新評估目標國

## Instructions

- 時間軸要具體到「月」，不是只說「提前一年準備」
- 每個 checklist 項目要有明確的「完成標準」
- 推薦信至少提前 3 個月跟教授/主管溝通
- SOP 要針對每所學校客製化，不是一稿多投
- 作品集要展示「與目標學程相關」的專案

### 標準申請時間軸（以 2028 Q1 入學為例）

```
2026 Q3 (Now)
├── 開始準備 IELTS/TOEFL
├── 研究目標學校清單
└── 盤點現有專案/作品

2026 Q4
├── 第一次 IELTS 考試（目標 7.0+）
├── 開始寫 SOP 初稿
└── 聯繫推薦人（教授 + 主管）

2027 Q1
├── IELTS 出分（未達標則 Q2 重考）
├── SOP 定稿（每校客製版）
├── CV/Resume 更新
└── 作品集整理

2027 Q2
├── 第一輪申請截止（UK early round）
├── 推薦信上傳
└── 申請費繳納

2027 Q3
├── 第二輪申請截止（SG main round）
├── 面試準備（部分學校有）
└── 獎學金申請

2027 Q4
├── Offer 陸續發放
├── 比較 offer（用 evaluating-career-roi）
├── 繳留位費
└── 開始簽證申請

2028 Q1
├── 簽證核發
├── 機票/住宿安排
├── 投資帳戶調整（提領學費）
└── 入學！
```

### Checklist template

```markdown
## 申請進度追蹤：<入學年份>

### 考試
- [ ] IELTS/TOEFL 報名
- [ ] IELTS 第一次考試（目標：7.0+）
- [ ] IELTS 成績達標（若未達：重考日期___）
- [ ] GRE（如需要）

### 文件
- [ ] CV/Resume（最終版）
- [ ] SOP 初稿
- [ ] SOP — <School A> 客製版
- [ ] SOP — <School B> 客製版
- [ ] 作品集/Portfolio
- [ ] 財力證明

### 推薦信
- [ ] 推薦人 1：<Name>（已聯繫 / 已寄出 / 已上傳）
- [ ] 推薦人 2：<Name>（已聯繫 / 已寄出 / 已上傳）

### 申請
- [ ] <School A>（截止：___，已送出：Y/N）
- [ ] <School B>（截止：___，已送出：Y/N）
- [ ] <School C>（截止：___，已送出：Y/N）

### 結果
- [ ] <School A>（Offer / Waitlist / Reject）
- [ ] <School B>（Offer / Waitlist / Reject）
- [ ] 最終決定：___
- [ ] 留位費繳納
- [ ] 簽證申請
```

## Error handling
- 若 IELTS 未達標且已無重考機會，評估是否接受 pre-sessional 語言課
- 若所有學校都被拒，啟動「延後一年 + 補強」計劃
- 若 offer 截止日衝突，優先繳留位費最低的學校
