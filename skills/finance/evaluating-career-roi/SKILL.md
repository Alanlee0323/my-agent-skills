# Evaluating Career ROI

## When to use this skill
- 當使用者問「Imperial vs NUS 哪個 ROI 高」
- 當需要量化出國念書的機會成本 vs 職涯溢價
- 當需要比較「留台 vs 出國」的長期財務影響
- 當需要在多個 offer 之間做財務面的決策

## Workflow

### 1. Plan
- 確認比較對象（學校 A vs B、國家 A vs B、留台 vs 出國）
- 確認投資人參數（現有資產、月存能力、投資報酬率假設）
- 確認時間範圍（5 年 ROI、10 年 ROI、30 年職涯終值）

### 2. Collect
- 學費 + 生活費（from `researching-programs`）
- 目標市場稅後薪資（from `scanning-job-market`）
- 有效稅率（各國所得稅 + 社會保險）
- 當地生活成本（扣除後的可投資金額）
- 匯率假設（使用當前匯率 + ±10% sensitivity）

### 3. Calculate
- **直接成本**：學費 + 生活費 + 簽證 + 機票
- **機會成本**：提領投資的錯失複利 + 停止月存的期間
- **總機會成本** = 直接成本 + 錯失複利 + 停止投入
- **職涯溢價**：(海外稅後可投資 - 留台稅後可投資) × 年數 × 複利
- **ROI** = 職涯溢價 / 總機會成本

### 4. Sensitivity
- 匯率 ±10%
- 薪資成長率 ±3%
- 投資報酬率 12% / 15% / 17% / 20%
- 留學後回台 vs 留在當地

## Instructions

- 所有金額統一換算為 TWD 做最終比較
- 稅率必須查實際累進稅率，不可用「大概 30%」
- 機會成本要包含複利效果，不是只算「少存了多少」
- 職涯溢價至少算 10 年和 30 年兩個時間範圍
- 結論要明確給出「划算 / 不划算 / 邊際」的判斷

### 計算公式

```
總機會成本 = 學費生活費 + (提領金額 × (1+r)^n - 提領金額) + (月存 × 停學月數)
職涯溢價(N年) = Σ[(海外月可投資 - 台灣月可投資) × 12 × (1+r)^(N-i)] for i=1..N
ROI = 職涯溢價 / 總機會成本
```

## Error handling
- 若稅率資料不確定，列出最低和最高估計的 range
- 若匯率波動大（>15%），在結論中特別標註匯率風險
- 若使用者的投資報酬率假設 >20%，提醒這是高度樂觀估計
